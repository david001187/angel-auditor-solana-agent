# 🤖 AngeL v11: Autonomous Solana Auditor

AngeL is an AI agent that autonomously audits the Solana blockchain to identify institutional movements and market risks.

## 🧠 Why it is Autonomous
- **Self-Planning**: Operates in 90-second cycles without human input.
- **Decision Making**: Autonomously filters "market noise" to find real whale transactions.
- **Automated Reporting**: Generates CSV audit logs independently.

## ⛓️ Solana Integration
- Uses **Solana RPC** for real-time indexing.
- Provides **Solscan** verification links for every audit.

## 🛠️ Execution
1. `pip install -r requirements.txt`
2. `python3 angel_auditor.py`
